# Multiplayer Limit Break

一个 Slay the Spire 2 多人模式限制突破 Mod。

## 用途

将多人模式人数上限从原版 4 人提高到最多 16 人，并配套调整大厅序列化、房间布局和多人难度缩放。

该功能默认关闭，以保持和原版客户端的兼容性。需要使用时，可在 RitsuLib 设置中开启。

## 依赖

- [STS2-RitsuLib](https://github.com/OLC-0/sts-2-ritsulib) 0.4.13 或更高版本

## Reference

本项目的基本思路来自 [Rain156/sts2-RMP-Mods](https://github.com/Rain156/sts2-RMP-Mods)。

## English

A Slay the Spire 2 mod that raises the multiplayer player limit from the vanilla 4 players to up to 16 players, with related lobby serialization, room layout, and multiplayer difficulty-scaling adjustments.

The limit break is disabled by default to keep compatibility with vanilla clients. Enable it from the RitsuLib settings when needed.

Requires [STS2-RitsuLib](https://github.com/OLC-0/sts-2-ritsulib) 0.4.13 or later.

The basic idea for this project comes from [Rain156/sts2-RMP-Mods](https://github.com/Rain156/sts2-RMP-Mods).
